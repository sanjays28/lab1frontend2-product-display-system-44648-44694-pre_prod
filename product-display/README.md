# Product Display Component

A simple, responsive product display component built with HTML and CSS.

## Project Structure
```
product-display/
├── src/
│   ├── css/
│   │   └── styles.css
│   ├── images/
│   └── index.html
└── README.md
```

## Setup
1. Clone the repository
2. Open `src/index.html` in a web browser

## Requirements
- Web browser with HTML5 and CSS3 support
